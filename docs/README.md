# Dokumentacja projektu Retro Neon

- `styleguide.md` - styleguide projektu (kolory, typografia, komponenty, dostępność).
- `deploy-cloudflare-pages.md` - instrukcja wdrożenia demo na Cloudflare Pages (SPA fallback).
- `final-report.md` - raport końcowy do oceny.
- `ux-tests.md` - pełny raport testów UX.
- `ux-iteration-1.md` - pierwsza iteracja UX (Profil/Osiągnięcia/Statystyki).
- `ux-iteration-2.md` - druga iteracja UX (Hub/Ranking/A-B menu).
- `a11y-audit.md` - audyt dostępności i wdrożone poprawki.
- `smoke-test.md` - finalny manualny smoke test.
- `checklist.md` - checklista oddania projektu.
- `screens/README.md` - galeria i instrukcja generowania zrzutów ekranu.
